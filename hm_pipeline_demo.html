README — Reproducing This Run

This demo chains four production modules into a single observable pipeline. All data shown was generated by running the modules against the Japan STR Art. 11(1)(iii) test vector with a deliberate two-agent disagreement on threshold_value (10 vs 15 minutes).

Prerequisites: Python 3.8+ · All five module files in the same directory.

pip install nothing  # zero external dependencies

# Files required in same directory:
hm_intake_screener.py
hm_consensus.py
hm_telemetry.py
hm_adversarial_suite.py  (optional — for Stage 03)

Run end-to-end:

python hm_pipeline_demo.py

Modules:

HM-SCREEN-001 — Intake Screening · HM-ADVERSARIAL-001 — Adversarial Suite · HM-CONSENSUS-001 — Consensus Resolution · HM-TELEMETRY-001 — Telemetry
